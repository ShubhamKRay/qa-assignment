## 📄 Assignment Overview

This repository contains my submission for the QA Engineer Intern assignment at SimplifyMoney.

The assignment focuses on testing core functionalities of the application and identifying real-world issues based on user behavior.

---

## 🔍 Scope of Testing

The following user flows were tested:

* Buy Flow
* Sell Flow
* Gift Flow
* SIP (Systematic Investment Plan) Flow

---

## 🧪 Testing Approach

* Performed manual testing on real application flows
* Used both valid and invalid inputs
* Tested edge cases such as multiple login attempts and OTP handling
* Focused on identifying functional bugs rather than UI/UX issues
* Re-tested scenarios to ensure consistency

---

## 🐞 Key Findings

* OTP delay observed during multiple login attempts
* User session handling issue (automatic logout on user switch without notification)
* Biometric authentication inconsistency
